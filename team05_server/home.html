Hello and welcome to our team streaming platform !

To upload a new video acces the "new.php" page and provide a title and a description of the video in JSON. The page will return your video ID
The parts of the video should be named video_ID-part_number.mp4 and can be uploaded using the "upload.php" page (the "upload" page can be used to manually upload it)

To view all the available videos access the "video-info-DASH.php" page it will return a JSON containing the video title, url of the DASH playlist and description of all the available videos.
The url returned are the url of the mpd playlist

If you want to play the videos form Safari go to the page list-safari.php which will display the available videos, this page also provides a link to the m3u8 playlist which make them playable on a Safari web browser.

This server also contains:

  • A mpd test file name mpdGenerator.mpd
  • A mp4 video for testing Star-Wars.mp4
  • An id.php page which gives you some infos about yourself that can be used to test if the server is online.
  • A dash.html page which uses an outside tool to play a given mpd file.
  • A "video-info-HLS.php" page which does the same thing as "video-info-DASH" but with the url of the m3u8 playlist
